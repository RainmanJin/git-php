返回码说明

{
    success:当为true时说明请求成功，false时为失败
    message:一般是错误信息，当 success 为 true 时，才可能为""
    token:"74hksdl84xhjuyqwet", //也可能为 null
    result://接口需求的具体数据
    注：所有请求默认post方式，除了登录忘记密码获取验证码接口其余接口默认全部传入token
    
}
